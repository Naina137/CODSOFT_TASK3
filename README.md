# CODSOFT_TASK3
# Iris Flower Classification
# Project Overview
This project demonstrates the implementation of a Machine Learning model to classify Iris flowers into three species using the Decision Tree algorithm. The model is trained on the Iris dataset available in the Scikit-learn library and achieves high classification accuracy.
# Objectives
- Load and explore the Iris dataset.
- Perform data preprocessing and analysis.
- Visualize the relationship between flower features.
- Train a Decision Tree Classifier.
- Evaluate the model using test data.
- Compare actual and predicted outputs.
# Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
# Dataset
The project uses the Iris dataset provided by the Scikit-learn library. The dataset contains 150 flower samples belonging to three species:
- Iris Setosa
- Iris Versicolor
- Iris Virginica
Each sample contains four features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
# Machine Learning Model
**Algorithm:** Decision Tree Classifier
# Project Workflow
1. Import required libraries.
2. Load the Iris dataset.
3. Convert the dataset into a Pandas DataFrame.
4. Perform exploratory data analysis.
5. Visualize feature relationships.
6. Split the dataset into training and testing sets.
7. Train the Decision Tree model.
8. Predict flower species.
9. Evaluate model performance.
10. Compare actual and predicted values.
# Results
The Decision Tree Classifier achieved an accuracy of **100%** on the test dataset, demonstrating excellent performance for this classification task.
